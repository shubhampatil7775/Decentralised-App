#INSTALLATIONS
1. Ganache(local blockchain)
2. Metamask Extension for chrome
3. node js

#STEPS
1. Download repo zip
2. Open cmd from the repo directory location type command "npm install"
3. Open ganache and copy private key of any account
4. Add this private key to metamask extension in add account section
5. Connect that account to local network
6. (After 2nd step)in cmd type npm start
7. Project start on localhost 3000
